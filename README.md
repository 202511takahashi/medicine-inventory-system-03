# 薬品在庫管理システム

## アプリ概要

このアプリは薬品の在庫管理を行うWebアプリです。  
薬品の在庫数、使用期限、状態をまとめて管理し、在庫不足や期限切れの見落としを防ぐことを目的としています。

就職用ポートフォリオとして開発しています。

## 機能概要

このアプリでは、薬品在庫管理に必要な基本操作と、安全管理を意識した確認機能を実装しています。

- 薬品一覧表示
  - 登録済みの薬品を一覧で確認できます。
- 薬品登録
  - 新しい薬品情報をフォームから登録できます。
- 薬品編集
  - 既存の薬品情報を編集して更新できます。
- 薬品削除
  - 一覧画面から不要な薬品データを削除できます。
- 薬品詳細画面
  - 1件ごとの薬品情報を見やすく確認できます。
- 検索機能
  - 薬品名・カテゴリで絞り込み検索ができます。
- カテゴリ並び替え
  - カテゴリ昇順・降順で一覧を並び替えできます。
- 在庫不足アラート
  - 在庫数が少ない薬品をひと目で確認できます。
- 使用期限アラート
  - 使用期限が1か月以内の薬品を期限注意として表示します。
- 期限切れ表示
  - 使用期限が過ぎた薬品を期限切れとして強調表示します。
- 状態表示
  - 正常・在庫不足・期限注意・期限切れを状態バッジで確認できます。
- カテゴリ色分け
  - カテゴリごとに色分けして見やすく表示しています。
- ダッシュボード風UI
  - 一覧画面を見やすいダッシュボード風レイアウトにしています。
- 集計カード
  - 総薬品数・在庫不足・期限注意・期限切れを上部カードで確認できます。
- トップ画面UI
  - イラスト付きのトップ画面から一覧や登録画面へ遷移できます。
