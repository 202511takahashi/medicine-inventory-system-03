# 薬品在庫管理システム

## アプリ概要

このアプリは薬品の在庫管理を行うWebアプリです。  
薬品の在庫数、使用期限、入出庫情報を管理し、在庫不足や期限切れを防ぐことを目的としています。

就職用ポートフォリオとして開発しています。

## 使用技術

- Java
- Spring Boot
- JSP
- HTML
- CSS
- MySQL
- Git / GitHub
