# 薬品在庫管理システム

ポートフォリオ用に作成している、薬品の在庫を管理するための Web アプリケーションです。  
現在は Spring Boot と JSP を使った初期プロジェクト構成のみを作成しています。

## 概要

このアプリは、薬品の在庫状況を一覧で確認し、登録・編集・削除・検索などを行えるようにすることを目指しています。  
今回は今後の機能追加がしやすいように、シンプルなトップ画面と基本構成を用意しました。

## 使用技術

- Java
- Spring Boot
- JSP
- HTML
- CSS
- MySQL
- Maven

## 現在の実装状況

- Spring Boot の初期プロジェクト構成を作成
- JSP を表示するための設定を追加
- `/` にアクセスするとトップ画面を表示
- シンプルなトップページと CSS を作成

## 今後追加予定の機能

- 薬品一覧
- 薬品登録
- 編集
- 削除
- 検索
- 在庫アラート

## ディレクトリ構成

```text
src/main/java/com/example/medicineinventory
src/main/resources
src/main/resources/static/css
src/main/webapp/WEB-INF/jsp
```
